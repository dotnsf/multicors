# Multi CORS


## Overview

複数のオリジンに対応する CORS 実装


## Environment Values

- `CORS`

  - REST API( `GET /ping` ) へのクロスアクセスを許可するオリジンをカンマ区切りで指定

    - 例１: CORS=http://localhost:8000,https://xxxx.herouapp.com

    - 例２: CORS=*
